# Image viewer on Rust

Программа открывает файл, указанный в первом параметре запуска, иначе открывает заглушку.

**Build dep:**
- cargo = ^1.57

**Linux dep:**
- X11
- Xext
- Xinerama
- Xcursor
- Xrender
- Xfixes
- Xft
- fontconfig
- pango-1.0
- pangoxft-1.0
- gobject-2.0
- cairo
- pangocairo-1.0
